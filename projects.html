Senior Design

Tools:: 3D Printing, Arduino, Wiring, SolidWorks

Description: The goal of this project was to capture solar energy from the sun and store it in phase change material (PCM) for a period of 24 hours. This was achieved by developing a system of tubing that circulated water heated in the sun through a tank that contained PCM. After the PCM melted, the system was shutoff and left to sit until the energy was needed.

For this project, I led 5 teammates in the yearlong process of planning, design, construction and implementation. For my part, I focused on the system piping, CAD model and control system.

Balancing Game

Tools:: Arduino, Wiring, Electronic Components

Description: I reached a point in my process of learning about the Arduino that I got bored with following tutorials. It dawned on me that the best way to learn was to decide on a project that interested me and hit the ground running with it. This was an attempt to make a basic balancing game in which the point is to use the joystick to keep the light moving inside the strip of LEDs. This is the first time I wrote code almost completely from scratch, taking many breaks to figure out how to do various aspects of the project.

Things Drawn Terribly Game

Tools:: JavaScript, KhanAcademy

Description: Similar to my Arduino game, I felt that I wanted to jump in and get started with JavaScript. Luckily, KhanAcademy provides an environment to easily create games so I decided to use this as the tool for my first JavaScript project. The goal of the game is to draw something using the limited quantity of colors, which are randomly generated, to draw the given prompts. The prompts were developed with three lists, each containing 100 words, one group of nouns, another of verbs, and the last of adjectives.

The game can be played at Khan Academy. (Note: There are some issues, which I believe are not because my code, that cause this game to malfunction on some computers.)
